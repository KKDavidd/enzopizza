# Assets

Helyezd ide a `hero-pizza.jpg` fájlt (a főoldal hero szekciójában megjelenő
pizza fotó). Ajánlott méret: kb. 900x1125px (4:5 arány), jó minőségű, közeli
pizza fotó hagyományos kőlapos sütésről.

Amíg nincs saját fotó feltöltve, a `js/info.js` a `settings/general.heroPhotoUrl`
mezőből is be tud tölteni egy képet a FireCMS-en keresztül feltöltött Firebase
Storage URL alapján — ez felülírja ezt a helyi fájlt.
